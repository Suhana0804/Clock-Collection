
# Stopwatch

This is a simple Stopwatch made using Python and Tkinter.

The stopwatch can:
- Start timing
- Stop timing
- Reset the timer

The time is displayed in:
- Hours
- Minutes
- Seconds

## Features

- GUI made using Tkinter
- Start button
- Stop button
- Reset button
- Exit button
- Real-time timer update

## Technologies Used

- Python
- Tkinter

## How to Run

Run the Python file:


python stopwatch.py
