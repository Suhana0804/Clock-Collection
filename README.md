# Clock Collection

This repository contains stopwatch and timer projects made using Python and Tkinter.
# Stopwatch

This is a simple Stopwatch made using Python and Tkinter.

The stopwatch can:
- Start timing
- Stop timing
- Reset the timer

The time is displayed in:
- Hours
- Minutes
- Seconds

## Features

- GUI made using Tkinter
- Start button
- Stop button
- Reset button
- Exit button
- Real-time timer update

## Technologies Used

- Python
- Tkinter

## How to Run

Run the Python file:


python stopwatch.py


## What I Learned

- Tkinter GUI basics
- Buttons and labels
- Functions in Python
- Using `after()` for timing
- Updating GUI elements
- Basic stopwatch logic

# Timer

This is a simple Timer application made using Python and Tkinter.

The user can set:
- Hours
- Minutes
- Seconds

After starting the timer, the countdown begins and the program displays a message when the time is over.

## Features

- GUI made using Tkinter
- Countdown timer
- User input for hours, minutes, and seconds
- Real-time timer update
- Message display when timer ends

## Technologies Used

- Python
- Tkinter

## How to Run

Run the Python file:


python timer.py

## What I Learned

- Tkinter GUI basics
- Entry boxes and labels
- Functions in Python
- Using `after()` for timing
- Countdown timer logic
- Updating GUI values dynamically
